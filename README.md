# IT Help Desk & Ticketing System Lab (Spiceworks)

## 📋 Project Overview
This project involved the implementation and management of a cloud-based **IT Service Management (ITSM)** environment using Spiceworks. I simulated a high-volume enterprise help desk, providing technical support for a diverse multi-departmental organization including **Law Enforcement (Sheriff's Dept)**, **Health & Human Services**, **Auditor/Finance**, and **Public Works**.

I successfully managed the full lifecycle of **15+ technical support tickets**, focusing on efficient intake, professional end-user communication, and detailed internal documentation.

## 🛠️ Skills & Competencies Demonstrated
* **ITSM Platform:** Spiceworks Cloud Help Desk.
* **Incident Management:** Full ticket lifecycle from **New** &rarr; **In Progress** &rarr; **Resolved**.
* **Prioritization:** Assigned urgency levels (Low/Medium/High) based on departmental impact (e.g., Pharmacy vs. General Admin).
* **Technical Documentation:** Authored **Internal Notes** for root cause analysis and a searchable knowledge base.
* **End-User Relations:** Crafted professional **Public Responses** to communicate resolutions clearly to non-technical staff.

## 🚀 Key Scenarios Resolved

### 1. Account & Access Management
* **Scenario:** Resolved "Account Locked" and "Password Reset" requests for field and office staff.
* **Action:** Verified user identity, cleared lockouts, and provided temporary credentials while identifying root causes (expired passwords on mobile devices).

### 2. Software & OS Troubleshooting
* **Scenario:** Addressed **Windows 11** "Update Loops" and critical **Blue Screen of Death (BSOD)** errors.
* **Action:** Diagnosed system crashes related to hardware/driver conflicts and provided manual override steps for stuck system updates.

### 3. Hardware & Peripheral Support
* **Scenario:** Managed diverse hardware failures, including **Pharmacy Label Printers**, **VoIP Desk Phones**, and "Sticky Key" issues on rugged field laptops.
* **Action:** Diagnosed physical malfunctions (fuser errors, connectivity drops), provided cleaning/maintenance guidance, and verified restored functionality.

### 4. Data Recovery & Connectivity
* **Scenario:** Handled urgent requests for "Deleted File Recovery" and "VPN/Hotspot Connectivity" failures.
* **Action:** Located missing directories on shared drives and resolved network timeout issues to restore access to mission-critical files.

## 📈 Lab Results
* **15+ Total Resolved Tickets:** Successfully cleared the support queue across four distinct departments.
* **Documentation Standards:** Every ticket contains a detailed internal log of the fix and a professional closing statement to the user.
* **SLA Compliance:** Prioritized emergency "Pharmacy" and "Sheriff" tickets to minimize downtime in high-stakes environments.
